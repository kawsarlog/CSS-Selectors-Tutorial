CSS Selectors Tutorial

Introduction

CSS selectors are powerful tools that allow you to target and style specific elements on a web page. This tutorial will provide an overview of different CSS selector rules and their usage.

Selector Rules

  • Type Selector(Tag):

    The type selector matches all elements of a given type (e.g. p for paragraphs, h1 for headings).

    Example: p selects all paragraphs on the page.

  • Class Selector(.):

    The class selector matches all elements with a specified class attribute.

    Example: .example-class selects all elements with class="example-class".

  • ID Selector(#):

    The ID selector matches the element with the specified ID attribute.

    Example: #example-id selects the element with id="example-id".

  • Attribute Selector:

    The attribute selector matches elements with a specified attribute and value.

    Example: [href="https://example.com"] selects all elements with href="https://example.com".

  • Selector List:

    A selector list allows you to apply styles to multiple selectors at once by separating them with commas.

    Example: h1, h2, h3 selects all h1, h2, and h3 elements on the page.

  • Descendant Combinator:

    The descendant combinator (space) selects all elements that are descendants of a specified parent element.

    Example: ul li selects all li elements that are descendants of ul elements on the page.

  • Child Combinator:

    The child combinator (>) Child Combinator:

    The child combinator (>) selects all elements that are direct children of a specified parent element.

    Example: ul > li selects all li elements that are direct children of ul elements on the page.

  • General Sibling Combinator:

    The general sibling combinator (~) selects all elements that are siblings of a specified element and appear after it.

    Example: h1 ~ p selects all p elements that are siblings of h1 elements and appear after them on the page.

  • Adjacent Sibling Combinator:

    The adjacent sibling combinator (+) selects the element that is the next sibling of a specified element.

    Example: h1 + p selects the p element that is the immediate sibling of h1 elements on the page.

  • Pseudo Classes:

    Pseudo-classes are used to select elements based on a certain state or condition.

    Example: a:hover selects the a element when it is being hovered over by the mouse cursor.

  • Pseudo Elements:

    Pseudo-elements are used to style a specific part of an element's content.

    Example: p::first-line selects the first line of text within p elements.

Some Addtional Rules

  • Containing Text of an Attribute(*):

    The attribute contains selector (*=) selects elements whose specified attribute contains a certain text.

    Example: input[id*='er-messa'] selects all input elements whose id attribute contains the text 'er-messa'.

  • Starting Text of an Attribute(^):

    The attribute starts with selector (^=) selects elements whose specified attribute starts with a certain text.

    Example: input[id^='user'] selects all input elements whose id attribute starts with the text 'user'.

  • Ending Text of an Attribute($):

    The attribute ends with selector ($=) selects elements whose specified attribute ends with a certain text.

    Example: input[id$='message'] selects all input elements whose id attribute ends with the text 'message'.

  • Comma Operator to Implement OR Operation(,):

    The comma operator (,) allows you to select multiple selectors and apply styles to elements that match any of the selectors.

    Example: p.red, p.blue selects all p elements with either the class red or the class blue.

  • Not Selector:

    The not selector (:not()) selects elements that do not match a specified selector.

    Example: p:not(.special) selects all p elements that do not have the class special.

  • Has Selector:

    The has selector (:has()) selects elements that contain elements matching a specified selector.

    Example: div:has(p) selects all div elements that contain p elements.

  • :first-of-type Selector:

    The :first-of-type selector selects the first element of its type among its siblings.

    Example: p:first-of-type selects the first p element among its sibling elements.

  • :last-of-type Selector:

    The :last-of-type selector selects the last element of its type among its siblings.

    Example: p:last-of-type selects the last p element among its sibling elements.

  • :nth-last-of-type(n) Selector:

    The :nth-last-of-type(n) selector selects the nth element of its type among its siblings, counting from the last element.

    Example: p:nth-last-of-type(2) selects the second-to-last p element among its sibling elements.

  • :nth-of-type(n) Selector:

    The :nth-of-type(n) selector selects the nth element of its type among its siblings.

    Example: p:nth-of-type(3) selects the third p element among its sibling elements.

  • :nth-child(n) Selector:

    The :nth-child(n) selector selects the nth child element among its siblings, regardless of the element type.

    Example: div:nth-child(odd) selects odd-numbered div elements among its sibling elements.
