Example: Selecting elements using the :not() selector

This example selects and applies styles to elements using the :not() selector to exclude elements with a specific class.

.example-element:not(.highlight) {
  background-color: #f9f9f9;
}

.example-element.highlight {
  background-color: yellow;
}
