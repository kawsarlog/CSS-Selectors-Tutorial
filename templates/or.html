Example: Selecting elements with comma-separated selectors

This example selects and applies styles to elements using comma-separated selectors to implement an OR operation.

.example-element.red, .example-element.blue {
  color: white;
  background-color: red;
}

.example-element.green, .example-element.yellow {
  color: white;
  background-color: green;
}
